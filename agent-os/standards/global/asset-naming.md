# Asset Naming (Content Team)

While this project focuses on C++ standards, content assets should also be consistent.

**Prefixes (suggested)**
- `SM_` Static Mesh, `SK_` Skeletal Mesh, `M_` Material, `MI_` Material Instance
- `T_` Texture, `MF_` Material Function, `P_` Niagara/Particle System
- `BP_` Blueprint, `WBP_` UMG Widget Blueprint, `AM_` Animation Montage

**Pattern**
`<Prefix><Descriptor>_<Qualifier>` e.g., `SM_Rock_Small_02`

- Keep names concise, readable, and searchable.
- Avoid spaces and punctuation; use UpperCamelCase separators.
